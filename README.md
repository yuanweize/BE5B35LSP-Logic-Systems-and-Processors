## 🎯 课程核心内容

### 1️⃣ 数字逻辑基础
- 布尔代数与逻辑门
- 卡诺图化简
- De Morgan定律
- Shannon展开

### 2️⃣ 组合逻辑电路
- 多路复用器/解复用器
- 编码器/解码器
- 加法器设计（全加器、+1加法器）

### 3️⃣ 时序逻辑电路
- RS锁存器（NOR/NAND实现）
- D触发器、T触发器
- 寄存器与计数器
- 状态机（Moore/Mealy）

### 4️⃣ VHDL硬件描述
- 并发语句 vs 顺序语句
- 信号 vs 变量
- 进程与敏感列表
- RTL视图分析

### 5️⃣ 计算机体系结构
- 分支预测（1位/2位预测器）
- Cache缓存（直接映射/组相联）
- 流水线基础
