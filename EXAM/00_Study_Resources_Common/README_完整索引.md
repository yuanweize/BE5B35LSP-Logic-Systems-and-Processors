## 🎯 四大必考题型 (2015-2025统计)

### 1️⃣ 有符号/无符号数 (100%出现率)
- 历年考点: 10位`10 0000 1111` → 527/-497
- 2025新变化: 运算结果截断 (124+125+126+127, 1023²)

### 2️⃣ 等价逻辑函数 (100%出现率)
- 历年固定答案: f4 ≡ f6 (都是 A XOR C)
- 2025新变化: 四变量函数，需要画4变量卡诺图

### 3️⃣ RS锁存器仿真 (100%出现率)
- 需要根据电路图分析输入输出
- 2025: 增加到5个时刻 (t0-t4)

### 4️⃣ 香农展开 (100%出现率)
- 公式: Q = Q̄·f0 + Q·f1
- 需要画f0和f1的卡诺图
