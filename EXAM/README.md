## 🎯 考试重点题型

### ⭐ 必考题型

1. **RS锁存器仿真** - 给定输入序列，写出输出值
2. **Shannon展开** - 函数分解为 f₀ 和 f₁
3. **等价逻辑函数** - 用卡诺图判断函数等价
4. **有符号/无符号数运算** - n位加法器结果计算

### 📊 常考题型

5. **Moore/Mealy自动机定义** - 六元组定义
6. **全加器/+1加法器设计** - 电路图绘制
7. **VHDL代码分析** - RTL视图绘制
8. **分支预测器** - 1位/2位预测器miss计算
9. **Cache计算** - 直接映射/组相联
